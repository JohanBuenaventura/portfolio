Reflecting on Backend Architecture

During week 4, I had to choose between a Monolithic structure or Microservices. Given the scale of this internship project, I decided to stick with a Monolithic approach but kept the code modular.

Fig 2. Monolithic structure chosen for project scope

I learned that over-engineering early on is the enemy of progress. Simple, clean code is better than complex, "clever" code.
